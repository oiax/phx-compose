# phx-compose

Docker を用いて Elixir/Phoenix アプリケーションの開発・学習を始めるための設定ファイル等のセット

## 構築される環境

* Debian 10 (Buster)
* Erlang/OTP 23.2
* Elixir 1.11.3
* Phoenix 1.5.7
* Node.js 14.15
* npm 6.14

## 必要なソフトウェア

* Docker 20 以上
* Docker Compose 1.27 以上
* Git 2.25 以上

## 動作確認済みのOS

* macOS Big Sur v11
* macOS Catalina v10.15
* Ubuntu Desktop 20.04 LTS (64-bit)
